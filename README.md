## Download

* [JumpG_v3.1_lite](https://github.com/Siddhanta-10/JumpG/releases/tag/3.1.2)

## Gameplay

<h1 align="center"><img src="Images/JumpG_v3.1_gameplay.gif"></img></h1>

<br>

1.Press SPACEBAR to jump and avoid obstacles
